# 美容サロン ランディングページ

## 概要
美容サロン向けのランディングページテンプレートです。初回限定特典を訴求し、LINE予約や電話予約を促進するデザインになっています。

## 特徴
- 初回限定50%OFF の特典訴求
- フェイシャル・ボディ・ネイルの各メニュー紹介
- お客様の声とFAQセクション
- LINE予約・電話予約の2つのCTA
- レスポンシブデザイン対応

# 商品レビュー ランディングページ

## 概要
商品レビュー・比較サイト向けのランディングページテンプレートです。商品の詳細レビューと購入促進を行うデザインになっています。

## 特徴
- 星評価システム
- インタラクティブなレビュー表示
- 商品比較テーブル
- 機能ハイライト
- 購入に関するFAQ
